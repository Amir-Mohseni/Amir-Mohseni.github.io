Research

Reasoning Router

Developed an automated router that predicts the optimal reasoning mode for each query, addressing the inefficiency of hybrid LLM systems. The system achieves performance within 5% of the best model's accuracy while reducing token usage by nearly 50% and improving response latency. View on Hugging Face

EduArena

Early engineer and core contributor to EduArena, a large-scale research crowd-sourcing platform for evaluating LLMs on educational tasks. Implemented automated pipelines for model selection and routing based on Bradley–Terry models, and developed reinforcement learning data generation systems using token- and answer-level entropy.

User–LLM Legal Interaction Analysis

MARBLE Research Project (Work in Progress): Developing a classifier to analyze how users seek legal advice from LLMs using real-world chatbot data. Created a taxonomy to categorize conversations by legal context and identify common patterns in user legal queries, with plans for publication submission.
